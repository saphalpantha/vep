# VEP Frontend

**VEP** is a comprehensive **Next.js frontend project** focusing on modern **UI/UX design** and interactive components. The project showcases a fully-featured dashboard, user account management, and modular reusable components for a professional web application interface.

### Key Features

- Interactive **Dashboard** with real-time UI updates.  
- User authentication pages: **Login** and **Signup**.  
- Component-based design with reusable **Cards, Activities, AppBar, and UI elements**.  
- Responsive and modern layout built with **Tailwind CSS**.  
- Modular structure for maintainability and scalability.  
- Designed with **professional UI/UX principles** for seamless user experience.

### Project Structure


/components # Reusable UI components (Cards, Activities, AppBar, etc.)
/pages # Next.js pages (Dashboard, Login, Signup, Index)
/styles # Component-specific styling using Tailwind CSS
/public # Static assets (logos, images, etc.)
/data.js # Sample or static data for UI components
